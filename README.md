# A Real-Time Vehicle Counting System for Detecting Traffic Surveillance

This is a Java web application built as an Eclipse dynamic web project. It is designed for real-time vehicle counting and traffic surveillance.

## Project Structure

- `src/` - Java source code for servlet controllers, utilities, login, and file handling
- `WebContent/` - Web application files including JSP pages, HTML, CSS, JS, and deployment descriptors
- `WebContent/WEB-INF/` - Web deployment descriptor and library dependencies
- `build/` - Generated compiled classes (ignored by git)

## Requirements

- Java JDK
- Apache Tomcat or another servlet container
- Eclipse IDE or VS Code with Java and Tomcat support

## How to Run

1. Import the project into Eclipse as an existing project.
2. Configure a Tomcat server runtime in Eclipse.
3. Run the project using `Run As > Run on Server`.

Alternatively, in VS Code:

1. Install Java extensions and Tomcat for Java.
2. Open the `Duplicationnew` folder.
3. Add your Tomcat installation in the Tomcat Explorer.
4. Deploy the project and run it on the server.

## Notes

- Compiled `*.class` files and the `build/` directory are excluded from version control.
- Source files and web resources are tracked in the repository.
